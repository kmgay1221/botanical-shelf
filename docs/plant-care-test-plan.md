## A. 単体テスト（npm test で自動実行・CI必須）

### A-1. lib/watering.ts
| # | ケース | 期待値 |
|---|---|---|
| U1 | 7/1に水やり、7月の間隔8日（大株） | 次回 7/9 |
| U2 | 1/25に水やり、1月の間隔30日 | 次回 2/24（水やり月の間隔を使用） |
| U3 | 当月がnull（断水） | 対象外・断水中表示情報（明け月）を返す |
| U4 | 断水明け判定: 12月null→1月値あり | 1/1 に dormancy_wake 対象 |
| U5 | 株サイズを中株→大株に変更後 | 次回計算が大株の間隔に切替 |
| U6 | 水やり記録が0件の株 | 登録日＋登録月の間隔 |
| U7 | 過去日付(6/20)で記録追加、最新は6/28 | 6/28基準のまま |
| U8 | 最新記録を削除 | 1つ前の記録基準に再計算 |
| U9 | 期日超過（次回が3日前） | 「今日やる」対象に含まれる |
| U10 | 実生苗（seedling）の全月値あり | 断水扱いにならない |
| U11 | 肥料: fertilizer_months外の月 | 肥料対象にならない |
| U12 | JST境界: UTC 7/5 16:00（=JST 7/6 1:00） | 「今日」がJSTの7/6として判定される |
